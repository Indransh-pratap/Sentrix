Sentrix is an AI-powered cybersecurity assistant and vulnerability awareness platform designed to help developers understand, analyze, and remediate common web application security issues.

It focuses on OWASP Top 10 vulnerabilities and provides structured, easy-to-understand technical explanations for secure development.
